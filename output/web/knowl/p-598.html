Given a path homotopy H: (I \times I,(0,0)) \to (X,x0), there is a unique path homotopy H̃: (I \times I,(0,0)) \to (X̃,x̃0) such that p \circ H̃ = H. Moreover, if H is a path homotopy between two paths f,g: (I,0) \to (X,x0), then H̃ is a path homotopy between the "lifts" f̃,g̃: (I,0) \to (X̃,x̃0) of f,g from the PLT.
