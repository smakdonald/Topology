Let X be a topological space. X is T1 if for any points a,b \in X with a \ne b, there are open sets U,V in X with a \in U, b ∉ U, a ∉ V, and b \in V. X is T2, or Hausdorff, if for any points a,b \in X with a \ne b, there are open sets U,V in X with U \cap V = \es, a \in U, and b \in V. X is T3 if X is T1 and for any point a \in X and closed set B in X with a ∉ B, there are open sets U,V in X with U \cap V = \es, a \in U, and B \sse V. X is T4 if X is T1 and for any closed sets A,B in X with A \cap B = \es, there are open sets U,V in X with U \cap V = \es, A \sse U, and B \sse V.
