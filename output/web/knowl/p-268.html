Let (X,\cT ) be a topological space, let A \sse X, and let p \in X. (1) The point p \in ClX(A) if and only if every open set in X that contains p intersects A. (2) If \cT = \cT (\cB ) is the topology generated by a basis \cB, then p \in ClX(A) if and only if every basis element containing p intersects A.
