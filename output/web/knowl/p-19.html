Suppose \(𝑋\) is a set. Then \(𝒯\) is a topology on \(𝑋\) if and only if \(𝒯\) is a collection of subsets of \(𝑋\) such that
  1. \(∅ ∈ 𝒯\text{,}\)
  2. \(𝑋 ∈ 𝒯\text{,}\)
  3. if \(𝑈 ∈ 𝒯\) and \(𝑉 ∈ 𝒯\text{,}\) then \(𝑈 ∩ 𝑉 ∈ 𝒯\text{,}\) and
  4. if \({𝑈𝛼}𝛼∈𝜆\) is any collection of sets of \(𝒯\text{,}\) then \(⋃𝛼∈𝜆 𝑈𝛼 ∈ 𝒯\text{.}\)
