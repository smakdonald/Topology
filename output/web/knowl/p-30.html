Let \(f: X → Y\) be a function. The function \(f\) is one-to-one (also called an injection and denoted \(f: X ↪ Y\)) if whenever \(x,x' ∈ X\) and \(f(x) = f(x')\) then \(x = x'\text{.}\) The function \(f\) is onto (also called a surjection and denoted \(f: X ↠ Y\)) if for every \(y\) in \(Y\text{,}\) there is an \(x\) in \(X\) with \(f(x) = y\text{.}\) The function \(f\) is a bijection if \(f\) is both one-to-one and onto. The function \(f\) is invertible if there is a function \(g:Y → X\) such that \(f ∘ g = 1_Y\) and \(g ∘ f = 1_X\text{.}\)
