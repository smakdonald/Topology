Theorem 1.2.12. PAN = “Preimages are nice”.

If \(f: X → Y\text{,}\) \(E ⊆ E' ⊆ Y\text{,}\) and \(E_α ⊆ Y\) for all \(α\text{,}\) then
  1. \(f\inv(E) ⊆ f\inv(E')\text{.}\)
  2. \(f\inv(∪_α Eα) = ∪_αf\inv(E_α)\text{.}\)
  3. \(f\inv(∩α E_α) = ∩_αf\inv(E_α)\text{.}\)
  4. \(f\inv(E' - E) = f\inv(E') - f\inv(E)\text{.}\)
