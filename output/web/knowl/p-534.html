Let X be a 2-dimensional PC CW complex and let p ∈ X be the basepoint. The inclusion map X(1) → X(2) induces a surjective homomorphism of fundamental groups whose kernel is generated by loops based at p in bijection with the 2-cells of X. More precisely: For each face f = eα2 of X define θα: I → X by θα := iX(1)X ∘ φα ∘ ω, where iX(1)X: X(1) → X is the inclusion map and ω:I → Sα1 is defined by ω(t) := (cos(2πt),sin(2πt)) for all t ∈ I. Let γα:I → X be a path from p to θα(0), and let wf := γα ∗ θα ∗ γαrev. Then π1(X,p) ≅ π1(X(1),p) / \langle{[wf]}\rangle^N.
