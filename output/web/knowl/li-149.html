Item 1. CW-CFBT = "CW-Continuous Function Building Thm".
Suppose that for all n ≥ 0 and α ∈ Jn, gα: Dn → Y is a continuous function. Suppose also that for all α ∈ Jm and β ∈ Jn with α ≠ β, whenever p ∈ Dαm, q ∈ Dβn, and Φα(p) = Φβ(q), then gα(p) = gβ(q). Then the collection {gα} induces a unique continuous function g: X → Y with g ∘ Φα = gα for all n ≥ 0 and α ∈ Jn.
