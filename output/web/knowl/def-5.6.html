Definition 5.1.4.

Let f,g:I \to X be paths in a space X.
  1. If f(1) = g(0), then their product, written f∗g, is the path in X from f(0) to g(1) defined by (f∗g)(s) := f(2s) for all s \in [0,1/2] and (f∗g)(s) := g(2s-1) for all s \in [1/2,1].
  2. The constant path at x_0, written cx_0, is the path (loop) in X defined by cx_0 := x_0 for all s \in I.
  3. The reverse of the path f is the path frev:I \to X defined by frev(s) := f(1-s) for all s \in I.
