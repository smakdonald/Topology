Let Z be a topological space, let A be an open set in Z, and let ∼ be an equivalence relation on Z satisfying the property that for each a ∈ A, the equivalence class of a satisfies [a] = \{a\}. If A has the subspace topology and Z/∼ has the quotient topology, then the function g: A → Z/∼ defined by g(a) := [a] for all a ∈ A is an embedding.
