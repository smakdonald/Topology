Theorem 1.2.13. IASN = “Images are sometimes nice”.

If \(f: X → Y, C ⊆ C' ⊆ X\text{,}\) and \(Cα ⊆ X for all α\text{,}\) then
  1. \(f(C) ⊆ f(C')\text{.}\)
  2. \(f(∪α Cα) = ∪α f(Cα)\text{.}\)
  3. \(f(∩α Cα) ⊆ ∩α f(Cα)\text{.}\) If moreover \(f\) is injective, then \(f(∩α Cα) = ∩α f(Cα)\text{.}\)
  4. \(f(C' - C) ⊇ f(C') - f(C)\text{.}\) If moreover f is injective, then \(f(C' - C) = f(C') - f(C)\text{.}\)
