Theorem 1.2.14.

Let \(f: X \to Y\text{.}\)
  1. If \(A \sse X\text{,}\) then \(A \sse f\inv(f(A))\text{;}\) if moreover \(f\) is injective then \(A = f\inv(f(A))\text{.}\)
  2. If \(B \sse Y\) then \(f(f\inv(B)) \sse B\text{;}\) if moreover \(f\) is surjective then \(f(f\inv(B)) = B\text{.}\)
