Another excursion into homological algebra:.

Definition 7.2.26.
A sequence of φ:A \to B, θ:B \to C of groups and homomorphisms is exact at B if Im φ = Ker θ. A sequence ⋯ Dn+1 \to Dn \to ⋯ is exact if it is exact at every group.
Lemma 7.2.27.
  1. A sequence φ:A \to B, θ:B \to 0 is exact if and only if φ is onto.
  2. A sequence φ:0 \to B, θ:B \to C is exact if and only if θ is one-to-one.
  3. If A \to B \to C \to D \to E is an exact sequence with homomorphisms φ:A \to B and ψ:D \to E satisfying Im φ = B and Ker ψ = 0, then C = 0.
