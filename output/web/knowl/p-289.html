Let \((X,\cT )\) be a topological space, let \(A \sse X\text{,}\) and let \(p \in X\text{.}\)
  1. The point \(p \in \text{Cl}_X(A)\) if and only if every open set in \(X\) that contains \(p\) intersects \(A\text{.}\)
  2. If \(\cT = \cT (\cB )\) is the topology generated by a basis \(\cB\text{,}\) then \(p \in \text{Cl}_X(A)\) if and only if every basis element containing \(p\) intersects \(A\text{.}\)
