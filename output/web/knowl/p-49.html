If \(f: X \to Y\text{,}\) \(E \sse E' \sse Y\text{,}\) and \(E_\alpha \sse Y\) for all \(\alpha \text{,}\) then
  1. \(f\inv(E) \sse f\inv(E')\text{.}\)
  2. \(f\inv(\cup_\alpha E\alpha ) = \cup_\alpha f\inv(E_\alpha )\text{.}\)
  3. \(f\inv(\cap_\alpha E_\alpha ) = \cap_\alpha f\inv(E_\alpha )\text{.}\)
  4. \(f\inv(E'\setminus E) = f\inv(E')\setminus f\inv(E)\text{.}\)
