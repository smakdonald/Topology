Let \(p: R \to S^1\) be defined by \(p(t) := (\cos(2\pi t), \sin(2\pi t))\) for all \(t \in R\text{.}\)
  • (PLT) Given a path \(f: (I,0) \to (S^1,(1,0))\text{,}\) there is a unique path \(f": (I,0) \to (R,0)\) such that \(p \circ f" = f\text{.}\)
  • (PHLT) Given a path homotopy \(H: (I \times I,(0,0)) \to (S^1,(1,0))\) there is a unique path homotopy \(H": (I \times I,(0,0)) \to (R,0)\) such that \(p \circ H" = H\text{.}\) Moreover, if \(H\) is a path homotopy between two paths \(f,g: (I,0) \to (S^1,(1,0))\text{,}\) then \(H"\) is a path homotopy between the "lifts" \(f",g": (I,0) \to (R,0)\) of \(f,g\) from the PLT.
