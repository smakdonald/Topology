Lemma 7.2.27.
  1. A sequence \(φ:A \to B\text{,}\) \(\theta :B \to 0\) is exact if and only if \(φ\) is onto.
  2. A sequence \(φ:0 \to B, \theta :B \to C\) is exact if and only if \(\theta \) is one-to-one.
  3. If \(A \to B \to C \to D \to E\) is an exact sequence with homomorphisms \(φ:A \to B\) and \(ψ:D \to E\) satisfying \(Im φ = B\) and \(Ker ψ = 0\text{,}\) then \(C = 0\text{.}\)
