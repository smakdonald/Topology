Let (X,𝒯) be a topological space, let A ⊆ X, and let p ∈ X. (1) The point p ∈ ClX(A) if and only if every open set in X that contains p intersects A. (2) If 𝒯 = 𝒯(ℬ) is the topology generated by a basis ℬ, then p ∈ ClX(A) if and only if every basis element containing p intersects A.
