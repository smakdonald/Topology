Definition 4.2.2.

Let Y be a subspace of a topological space X. A deformation retraction of X onto Y is a family of maps ft: X → X for all t ∈ I such that:
  1. f0 = 1X,
  2. f1(X) = Y,
  3. ft|Y = 1Y for all t ∈ I, and
  4. the function H: X × I → X defined by H(p,t) := ft(p), for all p ∈ X and t ∈ I, is continuous.
The function H is also called a deformation retraction from X to Y. The space Y is called a deformation retract of X, written X ⇝ Y.
