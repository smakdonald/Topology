Definition 3.36.

A metric on a set X is a function d: X × X → ℝ satisfying the following for all x,y,z ∈ X: (1) d(x,y) ≥ 0, and d(x,y) = 0 if and only if x = y. (2) d(x,y) = d(y,x). (3) d(x,z) ≤ d(x,y) + d(y,z). The pair (X,d) is a metric space, and for any x,y ∈ X the real number d(x,y) is the distance from x to y.
