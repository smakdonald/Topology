Let \(A_α\) for all \(α ∈ J\) be sets, let \(β ∈ J\text{,}\) and for each \(α ∈ J - \{β\}\text{,}\) let \(b_α ∈ A_α\text{.}\) The associated product inclusion map \(i_β: A_β → ∏_{α ∈ J} A_α\) is defined for each \(a ∈ A_β\) by \(i_β(a) := (a_α)_{α ∈ J}\text{,}\) where \(a_β := a\) and \(a_α := b_α for all α ∈ J - \{β\}\text{.}\)
