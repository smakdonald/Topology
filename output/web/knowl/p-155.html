Restrictions and extensions of continuous functions are continuous. That is: Let X and Y be topological spaces, let A be a subspace of X, and let B be a subspace of Y. (CC4) The inclusion i: A → X is continuous. (CC5) If f: X → Y is continuous, then f|A: A → Y is continuous. (CC6) If f: X → Y is continuous and f(X) ⊆ B, then f|B: X → B is continuous. (CC7) If f: X → B is continuous then f|Y: X → Y is continuous.
