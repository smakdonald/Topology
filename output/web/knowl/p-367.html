Let X be a compact space, let \sim be an equivalence relation on X, and let X/\sim be the quotient space. If Z is a T2 space and f: X \to Z is a continuous surjection satisfying [f(p) = f(q) if and only if p \sim q] for all p,q \in X, then the function g: X/\sim \to Z defined by g([p]) := f(p) for all [p] \in X/\sim is a homeomorphism.
