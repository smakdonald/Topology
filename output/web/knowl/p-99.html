Examples of topologies and topological spaces:
  1. Indiscrete topology on a set \(X: 𝒯_{ind} := \{∅, X\}\text{.}\)
  2. Discrete topology on a set \(X: 𝒯+{dis} := \{U | U ⊆ X\} = 𝒫(X) =\) the set of all subsets of \(X\text{.}\)
  3. Euclidean topology on \(ℝ^n: 𝒯_{Eucl} := \{U ⊆ ℝ | \text{for all} p ∈ U \text{there is an} r > 0 \text{and} q ∈ ℝ^n \text{such that} p ∈ B(q,r) ⊆ U \}\text{,}\) where \(B(q,r) = B_n(q,r) := \{x ∈ ℝ^n | d_{Eucl}(q,x) < r\}\) is the ball of radius \(r\) centered at q (and \(d_Eucl\) is Euclidean distance).
  4. Euclidean topology relative to a subset \(X ⊆ ℝn: 𝒯_{Eucl} sub := \{U ∩ X | U ⊆ ℝ^n and U is in 𝒯_{Eucl}\}\text{.}\)
  5. Finite complement topology on a set \(X: 𝒯_{fc} := \{U ⊆ X | X - U is a finite set\} ∪ \{∅\}\text{.}\)
  6. Included point topology on a set \(X\) with included point \(a ∈ X: 𝒯_{incl} := \{U ⊆ X | a ∈ U\} ∪ \{∅\}\text{.}\)
  7. Excluded point topology on a set \(X\) with excluded point \(a ∈ X: 𝒯_{excl} := \{U ⊆ X | a ∉ U\} ∪ \{X\}\text{.}\)
  8. Infinite ray topology on \(ℝ: 𝒯_{ray} := \{ (-∞,a) | a ∈ ℝ\} ∪ \{∅,ℝ\}\text{.}\)
  9. Preimage topology constructed on a set \(X\) from a function \(f:X → Y\) and a topology \(𝒯_Y\) on \(Y: 𝒯_f := \{f-1(U) | U ∈ 𝒯Y\}\text{.}\)
  10. Line with two origins with set \((ℝ - \{0\}) ∪ \{*,@\}\) and topology \(𝒯 := \{U | U ⊆ ℝ,- \{0\} and U ∈ 𝒯Eucl\} ∪ \{(U - \{0\}) ∪ \{*\} | U ⊆ ℝ, 0 ∈ U, and U ∈ 𝒯Eucl\} ∪ \{(U - \{0\}) ∪ \{@\} | U ⊆ ℝ, 0 ∈ U, and U ∈ 𝒯Eucl\} ∪ \{(U - \{0\}) ∪ \{*,@\} | U ⊆ ℝ, 0 ∈ U, and U ∈ 𝒯Eucl\}\text{.}\)
  11. On the finite set \(X = \{a,b,c\}: 𝒯 = \{ ∅, \{a\}, \{a,b\}, \{a,c\}, X\}\text{.}\)
