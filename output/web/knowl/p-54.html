If \(f: X \to Y, C \sse C' \sse X\text{,}\) and \(C_\alpha \sse X for all \alpha \text{,}\) then
  1. \(f(C) \sse f(C')\text{.}\)
  2. \(f(\cup \alpha C_\alpha ) = \cup \alpha f(C_\alpha )\text{.}\)
  3. \(f(\cap_\alpha C_\alpha ) \sse \cap_\alpha f(C_\alpha )\text{.}\) If moreover \(f\) is injective, then \(f(\cap_\alpha C_\alpha ) = \cap_\alpha f(C_\alpha )\text{.}\)
  4. \(f(C'\setminus C) \supseteq f(C')\setminus f(C)\text{.}\) If moreover \(f\) is injective, then \(f(C'\setminus C) = f(C')\setminus f(C)\text{.}\)
