Theorem 2.11.

Let X and Y be topological spaces, let A be a subspace of X, and let B be a subspace of Y.
  1. The inclusion i: A → X is continuous.
  2. If f: X → Y is continuous, then f|A: A → Y is continuous.
  3. If f: X → Y is continuous and f(X) ⊆ B, then f|B: X → B is continuous.
  4. If f: X → B is continuous then f|Y: X → Y is continuous.
