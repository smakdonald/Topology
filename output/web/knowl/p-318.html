  1. If ≈ is the smallest equivalence relation on I such that 0 ≈ 1, then I/≈ ≅ S1, with a homeomorphism that maps the equivalence class [s] to the point ( cos(2 π s) , sin(2 π s) ) for all s ∈ I.
  2. If ∼ is the smallest equivalence relation on I × I such that (s,1) ∼ (1,1) for all s ∈ I and (0,t) ∼ (1,1) ∼ (1,t) for all t ∈ I, then (I × I)/∼ ≅ D2, with a homeomorphism that maps the equivalence class [(1,1)] to the point (1,0) and maps [(s,0)] to (cos(2πs),sin(2πs)) for all s ∈ I.
