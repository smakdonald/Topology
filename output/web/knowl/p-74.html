Let \(∼\) be an equivalence relation on a set \(X\text{,}\) and let \(f: X → Y\) be a function satisfying the property that whenever \(x,x' ∈ X\) and \(x ∼ x'\) then \(f(x) = f(x')\text{.}\) Then:
  1. There is a well-defined function \(g:X/∼ → Y\) defined by \(g([x]) = f(x)\) for all \([x]\) in \(X/∼\text{;}\) that is, \(g ∘ q = f\text{,}\) where \(q\) is the equivalence map.
  2. If \(f\) is onto, then \(g\) is onto.
  3. If \(f\) also satisfies the property that whenever \(x,x' ∈ X\) and \(f(x) = f(x')\) then \(x ∼ x'\text{,}\) then \(g\) is one-to-one.
