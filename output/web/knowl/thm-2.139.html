Theorem 2.2.47.

Let Z be a topological space, let A be an open set in Z, and let \sim be an equivalence relation on Z satisfying the property that for each a \in A, the equivalence class of a satisfies [a] = \{a\}. If A has the subspace topology and Z/\sim has the quotient topology, then the function g: A \to Z/\sim defined by g(a) := [a] for all a \in A is an embedding.
