Let X be a CW complex with associated index sets Jn for the n-cells and characterstic maps Φα for each α ∈ Jn and each n, and let Y be a topological space.
  1. CW-CFBT = "CW-Continuous Function Building Thm".
    Suppose that for all n ≥ 0 and α ∈ Jn, gα: Dn → Y is a continuous function. Suppose also that for all α ∈ Jm and β ∈ Jn with α ≠ β, whenever p ∈ Dαm, q ∈ Dβn, and Φα(p) = Φβ(q), then gα(p) = gβ(q). Then the collection {gα} induces a unique continuous function g: X → Y with g ∘ Φα = gα for all n ≥ 0 and α ∈ Jn.
  2. CW-CFCT = "CW-Continuous Function Checking Thm".
    If f:X → Y is a function, then f is continuous iff every composition f ∘ Φα of f with an attaching map is continuous.
