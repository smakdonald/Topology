If f:X → Y is a surjection, then the quotient space (Y,𝒯f) is homeomorphic to the quotient space (X/∼,𝒯∼) induced by the equivalence relation on X defined by x ∼ x' if and only if f(x) = f(x') for all x,x' ∈ X; moreover, the homeomorphism g:X/∼ → Y satisfies g ∘ q = f, where q is the equivalence map.
