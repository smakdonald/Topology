A topological space \(X\) has a \(\delta\) -complex structure if and only if there is a collection of continuous maps \(\cC =\{\sigma \alpha :\delta n\alpha \to X\}\) such that:
  1. Each restriction \(\sigma \alpha |(0\delta n\alpha )\) is an embedding and each point of \(X\) is in the image of exactly one \(\sigma \alpha | (0\delta n\alpha )\text{.}\)
  2. Each \(\sigma \alpha \circ Fi(n\alpha )\) is a map \(\sigma \beta : \delta n\alpha -1 \to X \) in \(\cC\text{.}\)
  3. A subset \(A\) of \(X\) is open in \(X\) if and only if \(\sigma \alpha -1(A)\) is open in \(\delta n\alpha\) for all \(\sigma \alpha\) in \(\cC\text{.}\)
