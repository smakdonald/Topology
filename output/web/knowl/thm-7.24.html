Theorem 7.1.9.

A topological space X has a Δ-complex structure if and only if there is a collection of continuous maps 𝒞 = {σα:Δnα → X} such that: (1) Each restriction σα|(0Δnα) is an embedding and each point of X is in the image of exactly one σα| (0Δnα). (2) Each σα ∘ Fi(nα) is a map σβ: Δnα-1 → X in 𝒞. (3) A subset A of X is open in X if and only if σα-1(A) is open in Δnα for all σα in 𝒞.
