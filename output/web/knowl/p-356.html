  1. If X and Y are compact spaces, then the product space X \times Y is compact. (a') ("Tychonoff theorem") If X_\alpha is a compact space for all \alpha, then the product space \prod_\alpha X_\alpha is compact.
  2. Tube Lemma.
    If X and Y are spaces, p \in X, Y is compact, and N is an open set in the product space X \times Y containing {p} \times Y, then there is an open set W in X such that {p} \times Y \sse W \times Y \sse N.
