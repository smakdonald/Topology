Definition 1.2.24.
An equivalence relation \(∼\) on a set \(X\) is a subset \(S\) of \(X × X\) (where \((a,b) ∈ S\) is written \(a ∼ b\)) that satisfies the following for all \(a,b,c\) in \(X\text{:}\)
  1. \(a ∼ a\) (reflexive),
  2. \(a ∼ b\) implies \(b ∼ a\) (symmetric), and
  3. \(a ∼ b\) and \(b ∼ c\) implies \(a ∼ c\) (transitive).
The equivalence class of an element \(a\) of \(X\) is \([a] := \{b | b ∼ a\}\text{.}\) The notation \(X/∼\) denotes the set of equivalence classes, also called the quotient of \(X\) with respect to \(∼\text{.}\) The function \(q:X → X/∼\) defined by \(q(p) := [p]\) for all \(p ∈ X\) is called the equivalence map.
