Two paths f,g:I \to X in a space X are path homotopic, written f \simeq p g, if there is a continuous function H: I \times I \to X such that H(s,0) = f(s) and H(s,1) = g(s) for all s \in I, and H(0,t) = f(0) and H(1,t) = f(1) for all t \in I. The map H is called a path homotopy.
