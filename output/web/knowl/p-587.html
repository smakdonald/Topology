Let X be a CW complex.
  1. A subset B of X is closed in X if and only if B \cap X(n) is closed in X(n) for all n.
  2. If n \geq 0 and \beta \in Jn, then \text{Cl}_X(e\beta n) has nonempty intersection with at most finitely many open cell of X.
  3. A subset B of X is closed in X if and only if B \cap \text{Cl}_X(e\beta n) is closed in \text{Cl}_X(e\beta n) for all open cells e\beta n.
