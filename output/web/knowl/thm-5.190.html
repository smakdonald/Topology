Theorem 5.6.10. CW-HBT = "CW-Homeomorphism Building Thm".

Let X be a compact CW complex with associated index sets Jn for the n-cells and characterstic maps Φα for each α ∈ Jn and each n, and let Y be a T2 topological space. Suppose that for all n ≥ 0 and α ∈ Jn, gα: Dn → Y is a continuous function such that gα|Bn is injective. Suppose also that for all α ∈ Jm and β ∈ Jn with α ≠ β, (i) whenever p ∈ Dαm, q ∈ Dβn, and Φα(p) = Φβ(q), then gα(p) = gβ(q), and (ii) gα(Bαm) ∩ gβ(Bβn) = ∅. If Y = ∪n ≥ 0, α ∈ Jn gα(Bn), then the collection {gα} induces a unique homeomorphism g: X → Y with g ∘ Φα = gα for all n ≥ 0 and α ∈ Jn.
