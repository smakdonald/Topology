Let p: R → S1 be defined by p(t) := (cos(2π t), sin(2π t)) for all t ∈ R.
  • (PLT) Given a path f: (I,0) → (S1,(1,0)), there is a unique path f": (I,0) → (R,0) such that p ∘ f" = f.
  • (PHLT) Given a path homotopy H: (I × I,(0,0)) → (S1,(1,0)) there is a unique path homotopy H": (I × I,(0,0)) → (R,0) such that p ∘ H" = H. Moreover, if H is a path homotopy between two paths f,g: (I,0) → (S1,(1,0)), then H" is a path homotopy between the "lifts" f",g": (I,0) → (R,0) of f,g from the PLT.
