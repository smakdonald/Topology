(CC0)
  1. Let \(c\) be a real number. The function \(f:ℝ → ℝ\) defined by \(f(r) := cr\) for all \(r ∈ ℝ\) is continuous.
  2. The functions from \(ℝ^2\) to \(ℝ\) defined by addition, subtraction, and multiplication are continuous.
  3. The function \(g:ℝ × (ℝ - \{0\}) → ℝ\) defined by \(g(x,y) := x/y\) is continuous.
  4. A uniform limit of continuous functions from \(ℝ\) to \(ℝ\) is continuous; hence sine and cosine are continuous.
