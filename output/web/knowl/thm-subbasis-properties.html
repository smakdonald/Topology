Theorem 1.65.

Let 𝒮Y be a subbasis on a set Y.
  1. The basis ℬ(𝒮Y) generated by the subbasis 𝒮Y is a basis, and hence the topology 𝒯(𝒮Y) generated by the subbasis 𝒮Y is a topology.
  2. A function f: (X,𝒯X) → (Y,𝒯(𝒮Y)) is continuous if and only if for every subbasis element V ∈ 𝒮Y, the preimage f-1(V) is open in X.
