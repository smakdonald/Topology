Let p: (X̃,x̃0) → (X,x0) be a covering space.
  • PLT.
    Given a path f: (I,0) → (X,x0), there is a unique path f̃: (I,0) → (X̃,x̃0) such that p ∘ f̃ = f.
  • PHLT.
    Given a path homotopy H: (I × I,(0,0)) → (X,x0), there is a unique path homotopy H̃: (I × I,(0,0)) → (X̃,x̃0) such that p ∘ H̃ = H. Moreover, if H is a path homotopy between two paths f,g: (I,0) → (X,x0), then H̃ is a path homotopy between the "lifts" f̃,g̃: (I,0) → (X̃,x̃0) of f,g from the PLT.
