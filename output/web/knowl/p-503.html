Let A be a set and let R be a subset of F(A). The group presented by the presentation \langle A | R \rangle is the quotient group F(A)/\langle R \rangle^N. The set A is the set of generators, the set R is the set of relators, and the set of equations {r = 1 | r \in R} is the set of relations of the presentation.
