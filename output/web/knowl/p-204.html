Let X be a topological space, let ∼ be an equivalence relation on X, and let q:X → X/∼ be the quotient map. Let A be a subspace of X and let q(A) have the subspace topology in X/∼.
  1. The relation ≈ on A defined by [a ≈ a' if and only if a ∼ a' for all a,a' ∈ A] is an equivalence relation on A.
  2. If A = q-1(q(A)) and A is either an open or closed subset of X, then the the function q|Aq(A):A → q(A) is a quotient map and the function h: A/≈ → q(A) defined by h([a]) := [h(a)] for all [a] ∈ A/≈ is a homeomorphism.
