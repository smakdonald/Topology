Let X be a compact space, let ∼ be an equivalence relation on X, and let X/∼ be the quotient space. If Z is a T2 space and f: X → Z is a continuous surjection satisfying [f(p) = f(q) if and only if p ∼ q] for all p,q ∈ X, then the function g: X/∼ → Z defined by g([p]) := f(p) for all [p] ∈ X/∼ is a homeomorphism.
