Lemma 2.2.25.

Let \(A\) be a set, and for each \(\alpha\) let \(X_\alpha\) be a set. Then:
  1. Each function \(f: A \to \prod X_\alpha\) is a product function \(f = (f_\alpha )_\alpha\) such that \(f_\alpha : A \to X_\alpha\) is defined by \(f_\alpha := p_\alpha \circ f\) for each index \(\alpha\text{.}\)
  2. Given functions \(g_\alpha :A \to X_\alpha\) for each \(\alpha in J\text{,}\) the product function \(g := (g_\alpha )_{\alpha \in J}\) satisfies \(g_\alpha = p_\alpha \circ g\) for all \(\alpha\text{.}\)
