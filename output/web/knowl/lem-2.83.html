Lemma 2.2.22.

Let A be a set, and for each α let Xα be a set. Then: (a) Each function f: A → ∏ Xα is a product function f = (fα)α such that fα: A → Xα is defined by fα := pα ∘ f for each index α. (b) Given functions gα:A → Xα for each α in J, the product function g := (gα)α ∈ J satisfies gα = pα ∘ g for all α.
