Theorem 1.2.33.

Let \(X\) be a set. (1) The following are equivalent: (1a) \(X\) is finite. (1b) There is an onto function \(\{1,...,n\} ↠ X\) for some \(n ∈ ℕ\text{.}\) (1c) There is a one-to-one function \(X ↪ \{1,...,n\}\) for some \(n ∈ ℕ\text{.}\) (2) The following are equivalent: (2a) \(X\) is infinite. (2b) There is an onto function \(X ↠ ℕ\text{.}\) (2c) There is a one-to-one function \(ℕ ↪ X\text{.}\) (3) The following are equivalent: (3a) \(X\) is countable. (3b) There is an onto function \(ℕ → X\text{.}\) (3c) There is an injection \(X ↪ ℕ\text{.}\)
