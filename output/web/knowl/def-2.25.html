Definition 2.1.11.

Let \(X\) and \(Y\) be topological spaces. A function \(f: X → Y\) is a homeomorphism if \(f\) is a continuous bijection and \(f\inv\) is also continuous. The spaces \(X\) and \(Y\) are called homeomorphic, or topologically equivalent, if there is a homeomorphism from \(X\) to \(Y\text{;}\) in symbols this is written \(X ≅ Y\text{.}\)
