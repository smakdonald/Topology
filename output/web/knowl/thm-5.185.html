Theorem 5.6.9.

Let X be a CW complex with associated index sets Jn for the n-cells and characterstic maps Φ\alpha for each \alpha \in Jn and each n, and let Y be a topological space.
  1. CW-CFBT = "CW-Continuous Function Building Thm".
    Suppose that for all n ≥ 0 and \alpha \in Jn, g_\alpha : Dn \to Y is a continuous function. Suppose also that for all \alpha \in Jm and \beta \in Jn with \alpha \ne \beta, whenever p \in D\alpha m, q \in D\beta n, and Φ\alpha (p) = Φ\beta (q), then g_\alpha (p) = g\beta (q). Then the collection {g_\alpha } induces a unique continuous function g: X \to Y with g \circ Φ\alpha = g_\alpha for all n ≥ 0 and \alpha \in Jn.
  2. CW-CFCT = "CW-Continuous Function Checking Thm".
    If f:X \to Y is a function, then f is continuous iff every composition f \circ Φ\alpha of f with an attaching map is continuous.
