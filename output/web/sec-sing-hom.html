    Section 7.2 Singular Homology

    Subsection 7.2.1 Definitions and Induced Homomorphisms

    Definition 7.2.1.

    Let X be a topological space. A singular n-simplex of X is a continuous function σ:Δn → X.

    Definition 7.2.2.

    Let X be a topological space and let n ≥ 0. Let Sn be the set of singular n-simplices of X. The group of singular n-chains of X is Cnsing(X) = Cn(X) := ⊕singular n-simplices σ ℤ = ⊕singular n-simplices σ ℤσ. An element of Cnsing(X) is called a (singular) n-chain, and written ∑singular n-simplices σ mσσ, where each mσ ∈ ℤ and all but finitely many mσ's are 0.

    Definition 7.2.3.

    Let X be a topological space and let n ≥ 1. The n-th singular boundary map is the group homomorphism ∂nsing = ∂nX = ∂n: Cnsing(X) → Cn-1sing(X) defined by ∂n(∑singular n-simplices σ mσσ) := ∑singular n-simplices σ mσ∂n(1σ) and ∂n(1σ) := ∑i=0n (-1)i σ ∘ Fi(n). The 0-th singular boundary map is the group homomorphism ∂0sing = ∂0: C0sing(X) → 0.

    Lemma 7.2.4.

    If X is a topological space and n ≥ 1, then Im ∂n+1sing ⊆ Ker ∂nsing.

    Definition 7.2.5.

    Let X be a topological space and let n ≥ 0. The n-th singular homology group of X is the group Hnsing(X) = Hn(X) := Ker ∂nsing / Im ∂n+1sing.

    Example 7.2.6.

    Examples

    Proposition 7.2.7.

    If X is a space with exactly one point, then H0(X) = ℤ and Hn = 0 for all n ≥ 1.

    Theorem 7.2.8.

    If a space X has path components Xα, then Hnsing(X) = ⊕α Hnsing(Xα) for all n ≥ 0.

    Theorem 7.2.9.

    If X is a path-connected space, then H0sing(X) ≅ ℤ.

    Excursion into homological algebra.

    Definition 7.2.10.
    A chain complex, denoted C•, is a sequence ⋯ → Cn+1 → Cn → Cn-1 → ⋯ C0 → 0 with group homomorphisms ∂nC = ∂n: Cn → Cn-1 and ∂0C = ∂0: C0 → 0 satisfying Im ∂n+1 ⊆ Ker ∂n for all n. An element of Ker ∂n is called an n-cycle and an element of Im ∂n+1 is an n-boundary. For n ≥ 0, the n-th homology group of C• is Hn(C•) := Ker ∂nC / Im ∂n+1C.
    Definition 7.2.11.
    A chain map from a chain complex C• to a chain complex D•, written f♯:C• → D•, is a collection of group homomorphisms fn♯: Cn → Dn for all n ≥ 0 satisfying ∂nD ∘ fn♯ = fn-1♯ ∘ ∂nC for all n.
    Definition 7.2.12.
    Let f♯:C• → D• be a chain map. The n-th homology homomorphism induced by f♯ is the function fn∗: Hn(C•) → Hn(D•) defined by fn∗(γ + Im ∂n+1C) := fn♯(γ) + Im ∂n+1D.
    Lemma 7.2.13.
    If f♯:C• → D• is a chain map, then the n-th homology homomorphism fn∗ induced by f♯ is a well-defined group homomorphism.
    Definition 7.2.14.
    Two chain maps f♯,g♯: C• → D• are chain homotopic, written f♯ ≃ g♯, if there is a collection of group homomorphisms hn: Cn → Dn+1 satisfying gn - fn = ∂n+1D ∘ hn + hn-1 ∘ ∂nC for all n. The collection of functions {hn} is called a chain homotopy from f♯ to g♯.
    Proposition 7.2.15.
    If f♯,g♯: C• → D• are chain homotopic chain maps, then fn∗ = gn∗ for all n.
    Definition 7.2.16.
    A chain map f♯: C• → D• is a chain homotopy equivalence if there is a chain map g♯: D• → C• such that f♯ ∘ g♯ ≃ 1D• and g♯ ∘ f♯ ≃ 1C•.
    Proposition 7.2.17.
    If f♯: C• → D• is a chain homotopy equivalence, then fn∗: Hn(C•) → Hn(D•) is an isomorphism for all n.

    Theorem 7.2.18.

    If X is a Δ-complex, then HnΔ(X) ≅ Hnsing(X) for all n ≥ 0.

    Corollary 7.2.19.

    If X is a topological space with two Δ-complex structures X' and X'', then HnΔ(X') ≅ HnΔ(X'') for all n ≥ 0.

    Definition 7.2.20.

    Let X and Y be topological spaces and let f:X → Y be a continuous function, and let n ≥ 0. The n-th homology homomorphism induced by f is the homology homomorphism fn∗: Hn(X) → Hn(Y) induced by the (induced) chain map f♯: C•sing(X) → C•sing(Y) defined by fn♯(∑singular n-simplices σ of X mσσ) := ∑singular n-simplices σ of X mσ f ∘ σ.

    Lemma 7.2.21.

    If X and Y are topological spaces and f:X → Y is a continuous function, then the induced chain map f♯ is a well-defined chain map and hence the induced homology homomorphisms are well-defined homomorphisms.

    Proposition 7.2.22.

    1. If f: X → Y and g: Y → Z are continuous then (g ∘ f)n∗ = gn∗ ∘ fn∗ for all n.
    2. (1X)n∗ = 1Hn(X) for all n.
    3. If f and g are homotopic maps, then fn∗ = gn∗ for all n.

    Theorem 7.2.23.

    If X and Y are homotopy equivalent spaces (that is, X ≃ Y), then Hn(X) ≅ Hn(Y) for all n.

    Example 7.2.24.

    Examples

    Corollary 7.2.25.

    If X is a contractible space, then H0(X) ≅ ℤ and Hn(X) = 0 for all n ≥ 1.

    Subsection 7.2.2 Decomposing Spaces and Homology Groups

    Motivation: Want an SVK theorem for homology

    Another excursion into homological algebra:.

    Definition 7.2.26.
    A sequence of φ:A → B, θ:B → C of groups and homomorphisms is exact at B if Im φ = Ker θ. A sequence ⋯ Dn+1 → Dn → ⋯ is exact if it is exact at every group.
    Lemma 7.2.27.
    1. A sequence φ:A → B, θ:B → 0 is exact if and only if φ is onto.
    2. A sequence φ:0 → B, θ:B → C is exact if and only if θ is one-to-one.
    3. If A → B → C → D → E is an exact sequence with homomorphisms φ:A → B and ψ:D → E satisfying Im φ = B and Ker ψ = 0, then C = 0.

    Theorem 7.2.28. MV = Mayer-Vietoris Theorem.

    Suppose that X is a topological space with subspaces A,B such that X = Int(A) ∪ Int(B). Then there is an exact sequence ⋯ → Hn(A ∩ B) → Hn(A) ⊕ Hn(B) → Hn(X) → Hn-1(A ∩ B) → ⋯ → H0(X) → 0, such that each homorphism φn: Hn(A ∩ B) → Hn(A) ⊕ Hn(B) is given by φn := (iAB n∗,-iBA n∗) and each homomorphism ψn: Hn(A) ⊕ Hn(B) → Hn(X) is given by ψn := jA n∗ + jB n∗, where iAB: A ∩ B → A, iBA: A ∩ B → B, jA: A → X, and jB: B → X are inclusion maps.

    Example 7.2.29.

    Examples

    Proposition 7.2.30.

    Hi(Sn) ≅ ℤ for i = 0,n, and Hi(Sn) = 0 for all i ≠ 0,n.

    Yet another excursion into homological algebra.

    Definition 7.2.31.
    A short exact sequence of chain complexes is a sequence 0• → A• → B• → C• → 0• of chain complexes and chain maps such that for all n ≥ 0 the sequence 0 → An → Bn → Cn → 0 of groups and homomorphisms is exact.
    Theorem 7.2.32. Zig-zag Lemma.
    If 0• → A•, i♯:A• → B•, j♯:B• → C•, C• → 0 is a short exact sequence of chain complexes, then there is a (long) exact sequence ⋯ Hn(A) → Hn(B) → Hn(C) → Hn-1(A) → ⋯, with homomorphisms in∗:Hn(A) → Hn(B) and jn∗:Hn(B) → Hn(C).

    Proof of the MV Theorem.

    Definition 7.2.33.
    Let X be a topological space with subspaces A,B such that X = Int(A) ∪ Int(B). The chain complex C•(A+B) is defined by: Cn(A+B) := {∑ mσσ + ∑ kττ | each σ,τ is a singular n-simplex of X, each σ(Δn) ⊆ A and τ(Δn) ⊆ B, and each mσ, kτ ∈ ℤ} and ∂nA+B := ∂nX|Cn(A+B)Cn-1(A+B).
    Proposition 7.2.34.
    Let X be a topological space with subspaces A,B such that X = Int(A) ∪ Int(B). The sequence 0• → C•(A ∩ B) → C•(A) ⊕ C•(B) → C•(A+B) → 0•, with homomorphisms φ̃n: Cn(A ∩ B) → Cn(A) ⊕ Cn(B) given by φ̃n := (iAB n♯,-iBA n♯) and ψ̃n: Cn(A) ⊕ Cn(B) → Cn(A+B) given by ψ̃n := jA n♯ + jB n♯, is exact.
    Theorem 7.2.35. Small Chains Theorem.
    Let X be a topological space with subspaces A,B such that X = Int(A) ∪ Int(B). Then Hn(X) ≅ Hn(A+B) for all n.
