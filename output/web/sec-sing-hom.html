    Subsection 7.2.1 Definitions and Induced Homomorphisms

    Definition 7.2.1.

    Let \(X\) be a topological space. A singular \(n\)-simplex of \(X\) is a continuous function \(\sigma :\delta n \to X\text{.}\)

    Definition 7.2.2.

    Let \(X\) be a topological space and let \(n \geq 0\text{.}\) Let \(Sn\) be the set of singular \(n\)-simplices of \(X\text{.}\) The group of singular \(n\)-chains of \(X\) is \(Cnsing(X) = Cn(X) := \oplus singular n-simplices \sigma \Z = \oplus singular n-simplices \sigma \Z \sigma\text{.}\) An element of \(Cnsing(X)\) is called a (singular) \(n\)-chain, and written \(\sum singular n-simplices \sigma m\sigma \sigma\text{,}\) where each \(m\sigma \in \Z\) and all but finitely many \(m\sigma\)'s are \(0\text{.}\)

    Definition 7.2.3.

    Let \(X\) be a topological space and let \(n \geq 1\text{.}\) The \(n\)-th singular boundary map is the group homomorphism \(‚àÇnsing = ‚àÇnX = ‚àÇn: Cnsing(X) \to Cn-1sing(X)\) defined by \(‚àÇn(\sum singular n-simplices \sigma m\sigma \sigma ) := \sum singular n-simplices \sigma m\sigma ‚àÇn(1\sigma ) and ‚àÇn(1\sigma ) := \sum i=0n (-1)i \sigma \circ Fi(n)\text{.}\) The \(0\)-th singular boundary map is the group homomorphism \(‚àÇ0sing = ‚àÇ0: C0sing(X) \to 0\text{.}\)

    Lemma 7.2.4.

    If \(X\) is a topological space and \(n \geq 1\text{,}\) then \(Im ‚àÇn+1sing \sse Ker ‚àÇnsing\text{.}\)

    Definition 7.2.5.

    Let \(X\) be a topological space and let \(n \geq 0\text{.}\) The \(n\)-th singular homology group of \(X\) is the group \(Hnsing(X) = H_n(X) := Ker ‚àÇnsing / Im ‚àÇn+1sing\text{.}\)

    Example 7.2.6.

    Examples

    Proposition 7.2.7.

    If \(X\) is a space with exactly one point, then \(H_0(X) = \Z\) and \(Hn = 0\) for all \(n \geq 1\text{.}\)

    Theorem 7.2.8.

    If a space \(X\) has path components \(X_\alpha\text{,}\) then \(Hnsing(X) = \oplus \alpha Hnsing(X_\alpha )\) for all \(n \geq 0\text{.}\)

    Theorem 7.2.9.

    If \(X\) is a path-connected space, then \(H0sing(X) \cong \Z\text{.}\)

    Excursion into homological algebra.

    Definition 7.2.10.
    A chain complex, denoted \(C‚Ä¢\text{,}\) is a sequence \(\cdots \to Cn+1 \to Cn \to Cn-1 \to \cdots C0 \to 0\) with group homomorphisms \(‚àÇnC = ‚àÇn: Cn \to Cn-1\) and \(‚àÇ0C = ‚àÇ0: C0 \to 0\) satisfying \(Im ‚àÇn+1 \sse Ker ‚àÇn\) for all \(n\text{.}\) An element of \(Ker ‚àÇn\) is called an \(n\)-cycle and an element of \(Im ‚àÇn+1\) is an \(n\)-boundary.     Definition 7.2.11.
    A chain map from a chain complex \(C‚Ä¢\) to a chain complex \(D‚Ä¢\text{,}\) written \(f‚ôØ:C‚Ä¢ \to D‚Ä¢\text{,}\) is a collection of group homomorphisms \(fn‚ôØ: Cn \to Dn\) for all \(n \geq 0\) satisfying \(‚àÇnD \circ fn‚ôØ = fn-1‚ôØ \circ ‚àÇnC\) for all \(n\text{.}\)

    If \(f‚ôØ:C‚Ä¢ \to D‚Ä¢\) is a chain map, then the \(n\)-th homology homomorphism \(fn‚àó\) induced by \(f‚ôØ\) is a well-defined group homomorphism.
    Definition 7.2.14.
    Two chain maps \(f‚ôØ,g‚ôØ: C‚Ä¢ \to D‚Ä¢\) are chain homotopic, written \(f‚ôØ \simeq g‚ôØ\text{,}\) if there is a collection of group homomorphisms \(hn: Cn \to Dn+1 \)satisfying \(gn - fn = ‚àÇn+1D \circ hn + hn-1 \circ ‚àÇnC\) for all \(n\text{.}\) The collection of functions \(\{hn\}\) is called a chain homotopy from \(f‚ôØ\) to \(g‚ôØ\text{.}\)

    A chain map \(f‚ôØ: C‚Ä¢ \to D‚Ä¢\) is a chain homotopy equivalence if there is a chain map \(g‚ôØ: D‚Ä¢ \to C‚Ä¢\) such that \(f‚ôØ \circ g‚ôØ \simeq 1D‚Ä¢\) and \(g‚ôØ \circ f‚ôØ \simeq 1C‚Ä¢\text{.}\)

    Theorem 7.2.18.

    If \(X\) is a \(\delta\) -complex, then \(Hn\delta (X) \cong Hnsing(X)\) for all \(n \geq 0\text{.}\)

    Corollary 7.2.19.

    If \(X\) is a topological space with two \(\delta\) -complex structures \(X'\) and \(X''\text{,}\) then \(Hn\delta (X') \cong Hn\delta (X'')\) for all \(n \geq 0\text{.}\)

    Definition 7.2.20.

    Let \(X\) and \(Y\) be topological spaces and let \(f:X \to Y\) be a continuous function, and let \(n \geq 0\text{.}\) The \(n\)-th homology homomorphism induced by \(f\) is the homology homomorphism \(fn‚àó: H_n(X) \to H_n(Y)\) induced by the (induced) chain map \(f‚ôØ: C‚Ä¢sing(X) \to C‚Ä¢sing(Y)\) defined by \(fn‚ôØ(\sum singular n-simplices \sigma of X m\sigma \sigma ) := \sum singular n-simplices \sigma\) of \(X m\sigma f \circ \sigma\text{.}\)

    Lemma 7.2.21.

    If \(X\) and \(Y\) are topological spaces and \(f:X \to Y\) is a continuous function, then the induced chain map \(f‚ôØ\) is a well-defined chain map and hence the induced homology homomorphisms are well-defined homomorphisms.

    Proposition 7.2.22.

    1. If \(f: X \to Y\) and \(g: Y \to Z\) are continuous then \((g \circ f)n‚àó = gn‚àó \circ fn‚àó\) for all \(n\text{.}\)
    2. \((1X)n‚àó = 1H_n(X)\) for all \(n\text{.}\)
    3. If \(f\) and \(g\) are homotopic maps, then \(fn‚àó = gn‚àó\) for all \(n\text{.}\)

    Theorem 7.2.23.

    If \(X\) and \(Y\) are homotopy equivalent spaces (that is, \(X \simeq Y\)), then \(H_n(X) \cong H_n(Y)\) for all \(n\text{.}\)

    Example 7.2.24.

    Examples

    Corollary 7.2.25.

    If \(X\) is a contractible space, then \(H_0(X) \cong \Z\) and \(H_n(X) = 0\) for all \(n \geq 1\text{.}\)

    Definition 7.2.26.
    A sequence of \(œÜ:A \to B\text{,}\) \(\theta :B \to C\) of groups and homomorphisms is exact at \(B\) if \(Im œÜ = Ker \theta \text{.}\) A sequence \(\cdots Dn+1 \to Dn \to \cdots \) is exact if it is exact at every group.
    Lemma 7.2.27.
    1. A sequence \(œÜ:A \to B\text{,}\) \(\theta :B \to 0\) is exact if and only if \(œÜ\) is onto.

    Theorem 7.2.35. Small Chains Theorem.
    Let \(X\) be a topological space with subspaces \(A,B\) such that \(X = Int(A) \cup Int(B)\text{.}\) Then \(H_n(X) \cong H_n(A+B)\) for all \(n\text{.}\)
