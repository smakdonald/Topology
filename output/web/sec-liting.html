    Section 6.1 Definitions and Lifting

    Subsection 6.1.1 Definition

    Definition 6.1.1.

    A covering space of a topological space X is a space X̃ together with a continuous function p: X̃ \to X satisfying: There is an open covering {U\alpha } of X such that for all \alpha the preimage p-1(U\alpha ) is a disjoint union of open sets in X̃, each of which is mapped (by the restriction of p) homeomorphically onto U\alpha. Each set U\alpha is called evenly covered.

    Subsection 6.1.2 Lifting Theorems

    Theorem 6.1.3. PPHLT = Path and Path Homotopy Lifting Theorems.

    Let p: (X̃,x̃0) \to (X,x0) be a covering space.
    • PLT.
      Given a path f: (I,0) \to (X,x0), there is a unique path f̃: (I,0) \to (X̃,x̃0) such that p \circ f̃ = f.
    • PHLT.
      Given a path homotopy H: (I \times I,(0,0)) \to (X,x0), there is a unique path homotopy H̃: (I \times I,(0,0)) \to (X̃,x̃0) such that p \circ H̃ = H. Moreover, if H is a path homotopy between two paths f,g: (I,0) \to (X,x0), then H̃ is a path homotopy between the "lifts" f̃,g̃: (I,0) \to (X̃,x̃0) of f,g from the PLT.

    Theorem 6.1.4.

    (LC+ULP = Lifting Criterion and Unique Lifting Property): Let p: (X̃,x̃0) \to (X,x0) be a covering space, let Y be a PC and LPC space, and let f: (Y,y0) \to (X,x0) be continuous.
    1. If f∗(\pi_1(Y,y0)) \sse p∗(\pi_1(X̃,x̃0)), then there is a unique continuous function f̃: (Y,y0) \to (X̃,x̃0) such that p \circ f̃ = f.
    2. If f∗(\pi_1(Y,y0)) ⊈ p∗(\pi_1(X̃,x̃0)), then there does not exist a continuous lift f̃: (Y,y0) \to (X̃,x̃0) such that p \circ f̃ = f.

    Theorem 6.1.5.

    If p: (X̃,x̃0) \to (X,x0) is a covering space, then the induced homomorphism p∗: \pi_1(X̃,x̃0) \to \pi_1(X,x0) is injective (that is, Ker p∗ = {1\pi_1(X̃,x̃0)}), and the image satisfies Im p∗ = p∗(\pi_1(X̃,x̃0)) = {[f] \in \pi_1(X,x0) | the lift f̃ of f at x̃0 is a loop in X̃}. Consequently, \pi_1(X̃,x̃0) is isomorphic to a subgroup of \pi_1(X,x0).

    Subsection 6.1.3 Application to group theory

    Corollary 6.1.6.

    1. For all n \in \N, the free group Fn is a subgroup of F2.
    2. The free group F(S) on a countably infinite set S is a subgroup of F2.

    Subsection 6.1.4 The number of sheets

    Definition 6.1.7.

    Let p: (X̃,x̃0) \to (X,x0) be a covering space such that X and X̃ are PC. The number of sheets of the covering space is |p-1({x0})|.

    Theorem 6.1.8. Lifting Correspondence.

    Let p: (X̃,x̃0) \to (X,x0) be a covering space such that X and X̃ are PC. Then:
    1. The Lifting Correspondence Function Φ: \pi_1(X,x0)/p∗(\pi_1(X̃,x̃0)) \to p-1({x0}) defined by Φ(p∗(\pi_1(X̃,x̃0))[f]) := f̃(1), where f̃ is the unique lift of the loop f to a path in X̃ starting at x̃0, is a well-defined bijection.
    2. For all points x' \in X, |p-1({x'})| = |p-1({x0})| = (the number of sheets of p) = the index of p∗(\pi_1(X̃,x̃0)) in \pi_1(X,x0).

    Subsection 6.1.5 Interactions with functions and constructions:

    Theorem 6.1.10.

    If p:X̃ \to X is a covering space, then p is an open map.

    Theorem 6.1.11.

    Let p:X̃ \to X be a covering space. If A is a subspace of X and Ã := p-1(A), then the restriction p|AÃ:Ã \to A is also a covering space.

    Theorem 6.1.12.

    Let pi:X̃i \to Xi be covering spaces for i \in {1,2}. Then q := (p1 \circ proj1,p2 \circ proj2) :X̃1 \times X̃2 \to X1 \times X2 [that is, q(S^1,S^2) := (p1(S^1),p2(S^2)) for all (S^1,S^2) \in X̃1 \times X̃2] is also a covering space.

    Subsection 6.1.6 Interactions with homeomorphism invariants:

    Corollary 6.1.13.

    Let p:X̃ \to X be a covering space. If X is a CW complex, then X̃ is a CW complex. Moreover, p maps open n-cells to open n-cells for all n.

    Theorem 6.1.14.

    Let p:X̃ \to X be a covering space satisfying the property that p-1({s}) is finite and nonempty for all s \in X. The space X̃ is compact and T2 if and only if the space X is compact and T2.
