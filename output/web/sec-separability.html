    Section 3.2 Separability

    Definition 3.17.

    A space (X,𝒯X) has a countable basis if the topology 𝒯X is generated by a basis that is a countable set.

    Theorem 3.18.

    A countable basis is a homeomorphism invariant.

    Theorem 3.19.

    The property of having a topology generated by a countable basis is preserved by subspaces and countable products.

    Definition 3.20.

    Let X be a topological space.
    1. X is T1 if for any points a,b ∈ X with a ≠ b, there are open sets U,V in X with a ∈ U, b ∉ U, a ∉ V, and b ∈ V.
    2. X is T2, or Hausdorff, if for any points a,b ∈ X with a ≠ b, there are open sets U,V in X with U ∩ V = ∅, a ∈ U, and b ∈ V.
    3. X is T3 if X is T1 and for any point a ∈ X and closed set B in X with a ∉ B, there are open sets U,V in X with U ∩ V = ∅, a ∈ U, and B ⊆ V.
    4. X is T4 if X is T1 and for any closed sets A,B in X with A ∩ B = ∅, there are open sets U,V in X with U ∩ V = ∅, A ⊆ U, and B ⊆ V.

    Convention 3.21.

    If X is a T1 space, then T3 is also called regular and T4 is also called normal.

    Exploration 3.22.

    1. Let 𝑋 be a space with the finite complement topology. Show that 𝑋 is 𝑇1.
    2. Show that ℝstd is Hausdorff.

    Exploration 3.23.

    1. Describe an example of a topological space that is 𝑇1 but not 𝑇2.
    2. Describe an example of a topological space that is 𝑇2 but not 𝑇3.
    3. Describe an example of a topological space that is 𝑇3 but not 𝑇4.

    Theorem 3.24.

    For each 1 ≤ i ≤ 4, the property Ti is a homeomorphism invariant.

    Proposition 3.25.

    A space X is T1 if and only if all one-points sets are closed in X.

    Theorem 3.26.

    For each 2 ≤ i ≤ 4, the property Ti implies the property Ti-1, but the converse fails.

    Theorem 3.27.

    T2 and T3 are preserved by subspaces and products.

    Remark 3.28.

    T2 and T3 are not preserved by quotients. T4 is not preserved by subspaces, products, or quotients.

    Theorem 3.29.

    A space X is T3 iff X is T1 and for all x ∈ X and open U in X with x ∈ U, there is an open set V in X with x ∈ V ⊆ ClX(V) ⊆ U.

    Theorem 3.30.

    A space X is T4 iff X is T1 and for all closed A and open U in X with A ⊆ U, there is an open set V in X with A ⊆ V ⊆ ClX(V) ⊆ U.

    Theorem 3.31.

    A continuous bijection from a compact space to a Hausdorff space is a homeomorphism.

    Theorem 3.32.

    Let X be a compact space, let ∼ be an equivalence relation on X, and let X/∼ be the quotient space. If Z is a T2 space and f: X → Z is a continuous surjection satisfying [f(p) = f(q) if and only if p ∼ q] for all p,q ∈ X, then the function g: X/∼ → Z defined by g([p]) := f(p) for all [p] ∈ X/∼ is a homeomorphism.

    Theorem 3.33.

    If X is a compact space, Y is a T2 space, and f: X → Y is a surjective continuous function, then f is a quotient map.

    Theorem 3.34.

    Compact Hausdorff spaces are T4. (The converse is not true.)

    Theorem 3.35.

    A T3 space with a countable basis is T4.
