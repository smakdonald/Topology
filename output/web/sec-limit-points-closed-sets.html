    Section 1.2 Limit Points and Closed Sets

    Definition 1.16. Limit Point.

    Let (𝑋, 𝒯) be a topological space, 𝐴 a subset of 𝑋, and 𝑝 a point in 𝑋. Then 𝑝 is a limit point of 𝐴 if and only if for each open set 𝑈 containing 𝑝, (𝑈 − {𝑝}) ∩ 𝐴 ≠ ∅. Notice that 𝑝 may or may not belong to 𝐴.
    Phrased differently, 𝑝 is a limit point of 𝐴 if every open set containing 𝑝 intersects 𝐴 at some point other than 𝑝 itself. Thus, the concept of a limit point gives us a way of capturing the idea of a point being “arbitrarily close” to a set without using the concept of distance. Instead, we use the idea of open sets in a topology.

    Exploration 1.17.

    Let 𝑋 = ℝ and 𝐴 = (1, 2). Verify that 0 is a limit point of 𝐴 in the indiscrete topology and the finite complement topology, but not in the standard topology nor the discrete topology on ℝ.

    Theorem 1.18.

    Suppose 𝑝 ∉ 𝐴 in a topological space (𝑋, 𝒯). Then 𝑝 is not a limit point of 𝐴 if and only if there exists a neighborhood 𝑈 of 𝑝 such that 𝑈 ∩ 𝐴 = ∅.

    Definition 1.19. Isolated Point.

    Let (𝑋, 𝒯) be a topological space, 𝐴 a subset of 𝑋, and 𝑝 a point in 𝑋. If 𝑝 ∈ 𝐴 but 𝑝 is not a limit point of 𝐴, then 𝑝 is an isolated point of 𝐴.

    Exploration 1.20.

    If 𝑝 is an isolated point of a set 𝐴 in a topological space 𝑋, show that there exists an open set 𝑈 such that 𝑈 ∩ 𝐴 = {𝑝}.

    Exploration 1.21.

    Give examples of sets 𝐴 in various topological spaces (𝑋, 𝒯) with
    1. a limit point of 𝐴 that is an element of 𝐴;
    2. a limit point of 𝐴 that is not an element of 𝐴;
    3. an isolated point of 𝐴;
    4. a point not in 𝐴 that is not a limit point of 𝐴.

    Definition 1.22. Closure.

    Let (𝑋, 𝒯) be a topological space and 𝐴 ⊂ 𝑋. Then the closure of 𝐴 in 𝑋, denoted \ov 𝐴 or Cl(𝐴) or Cl𝑋 (𝐴), is the set 𝐴 together with all its limit points in 𝑋.

    Definition 1.23. Closed Set.

    Let (𝑋, 𝒯) be a topological space and 𝐴 ⊂ 𝑋. The subset 𝐴 is closed if and only if \ov 𝐴 = 𝐴, in other words, if 𝐴 contains all its limit points.

    Exploration 1.24.

    1. Which sets are closed in a set 𝑋 with the discrete topology?
    2. Which sets are closed in a set 𝑋 with the indiscrete topology?
    3. Which sets are closed in a set 𝑋 with the finite complement topology?
    4. Which sets are closed in a set 𝑋 with the countable complement topology?

    Theorem 1.25. Closure of Closure.

    For any topological space (𝑋, 𝒯) and 𝐴 ⊂ 𝑋, the set \ov 𝐴 is closed. That is, for any set 𝐴 in a topological space, \ov 𝐴 = \ov\ov 𝐴.

    Theorem 1.26. Open and Closed Complements.

    Let (𝑋, 𝒯) be a topological space. Then the set 𝐴 is closed if and only if 𝑋 − 𝐴 is open.

    Theorem 1.27.

    Let (𝑋, 𝒯) be a topological space, and let 𝑈 be an open set and 𝐴 a closed subset of 𝑋. Then the set 𝑈 − 𝐴 is open and the set 𝐴 − 𝑈 is closed.

    Theorem 1.28. Properties of Closed Sets.

    Let (𝑋, 𝒯) be a topological space. Then:
    1. ∅ is closed.
    2. 𝑋 is closed.
    3. The union of finitely many closed sets is closed.
    4. Let {𝐴𝛼}𝛼∈𝜆 be a collection of closed subsets in (𝑋, 𝒯). Then ⋂𝛼∈𝜆 𝐴𝛼 is closed.

    Remark 1.29.

    Theorem 1.28 shows that an alternative way to build up a theory of topology would be to specify all subsets that are “closed,” and demand that they satisfy the above four properties.

    Exploration 1.30.

    Give an example to show that the union of infinitely many closed sets in a topological space may be a set that is not closed.

    Exploration 1.31.

    Give examples of topological spaces and sets in them that
    1. are closed, but not open;
    2. are open, but not closed;
    3. are both open and closed;
    4. are neither open nor closed.

    Warning 1.32.

    The words “closed” and “open” are not antonyms!

    Exploration 1.33.

    State whether each of the following sets are open, closed, both, or neither:
    1. In ℤ with the finite complement topology: {0, 1, 2}, {prime numbers}, {𝑛 ∣ |𝑛| ≥ 10}.
    2. In ℝ with the standard topology: (0, 1), (0, 1], [0, 1], {0, 1}, {1/𝑛 | 𝑛 ∈ ℕ}.
    3. In ℝ2 with the standard topology: {(𝑥, 𝑦) | 𝑥2 + 𝑦2 = 1}, {(𝑥, 𝑦) | 𝑥2 + 𝑦2 > 1}, {(𝑥, 𝑦) | 𝑥2 + 𝑦2 ≥ 1}.
    One way to think about the closure of a set is as the intersection of all the closed sets that contain it.

    Theorem 1.34. Characterization of Closure.

    For any set 𝐴 in a topological space 𝑋, the closure of 𝐴 equals the intersection of all closed sets containing 𝐴, that is, 𝐴 = ⋂ 𝐵⊃𝐴,𝐵∈𝒞 𝐵, where 𝒞 is the collection of all closed sets in 𝑋.

    Exploration 1.35.

    Pick several different subsets of ℝ, and find their closures in
    1. the discrete topology;
    2. the indiscrete topology;
    3. the finite complement topology;
    4. the standard topology.

    Theorem 1.36.

    Let 𝐴 and 𝐵 be subsets of a topological space 𝑋. Then:
    1. 𝐴 ⊂ 𝐵 implies \ov 𝐴 ⊂ \ov 𝐵.
    2. \ov 𝐴 ∪ 𝐵 = \ov 𝐴 ∪ \ov 𝐵.

    Exploration 1.37.

    Let {𝐴𝛼}𝛼∈𝜆 be a collection of subsets of a topological space 𝑋. Then is the statement ⋃ 𝛼∈𝜆 𝐴𝛼 = ⋃ 𝛼∈𝜆 𝐴𝛼 true?

    Definition 1.38. Interior.

    The interior of a set 𝐴 in a topological space 𝑋, denoted 𝐴∘ or Int(𝐴), is defined as: Int(𝐴) = ⋃ 𝑈⊂𝐴, 𝑈∈𝒯 𝑈. Points of Int(𝐴) are called interior points of 𝐴.

    Theorem 1.39. Characterization of Interior Points.

    Let 𝐴 be a subset of a topological space 𝑋. Then 𝑝 is an interior point of 𝐴 if and only if there exists an open set 𝑈 with 𝑝 ∈ 𝑈 ⊂ 𝐴.

    Exploration 1.40.

    Show that a set 𝑈 is open in a topological space 𝑋 if and only if every point of 𝑈 is an interior point of 𝑈.

    Definition 1.41. Boundary.

    The boundary of 𝐴, denoted Bd(𝐴) or 𝜕𝐴, is defined to be \ov 𝐴 ∩ \ov 𝑋 − 𝐴.

    Theorem 1.42.

    Let 𝐴 be a subset of a topological space 𝑋. Then Int(𝐴), Bd(𝐴), and Int(𝑋 − 𝐴) are disjoint sets whose union is 𝑋.

    Exploration 1.43.

    Pick several different subsets of ℝ, and for each one, find its interior and boundary using
    1. the discrete topology;
    2. the indiscrete topology;
    3. the finite complement topology;
    4. the standard topology.
